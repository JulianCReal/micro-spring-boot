Sobre MicroSpringBoot

Este proyecto demuestra capacidades de reflexión en Java para construir un mini-framework IoC.

Tecnologías

  • JavaReflection API — escanea clases y métodos en tiempo de ejecución
  • IoCAnnotations @RestController, @GetMapping, @RequestParam
  • HTTPServerSocket puro — sin dependencias externas
  • BuildMaven — gestión del ciclo de vida

Flujo

ComponentScanner escanea el classpath → encuentra @RestController → DispatcherHandler registra rutas @GetMapping → HttpServer recibe peticiones → invoca el método via reflexión.
